# OttoManagerPro-V2-App

Next.js App Router SaaS foundation scaffold.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui ready configuration
- Prisma setup folder

## Zero-Confusion Structure

- `app/`: routing, layouts, and entry pages
- `components/`: shared UI components
- `modules/`: feature modules (empty until implementation)
- `lib/`: shared utilities and service clients
- `prisma/`: Prisma setup (no models yet)
- `tests/`: automated test suites
